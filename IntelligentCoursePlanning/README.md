# 📚 智能课程规划系统

> 编程综合实践课程期末个人大项目

## 项目简介

智能课程规划系统是一个基于 C++17 的课程规划工具，能够根据培养方案、课程先修关系、学分要求和学生个人偏好，自动生成八学期的课程安排。

系统涵盖 1000 门课程、3000 个教学班，支持 12 个专业的培养方案，并提供 Web 前端可视化与 AI 选课顾问。

## 快速开始

```bash
# 1. 构建 C++ 后端
cmake -B build -S .
cmake --build build --config Release

# 2. 运行规划（自动测试 + JSON 导出）
build\Release\course_planner.exe

# 3. 启动前端服务器
cd frontend
python server.py
# 在浏览器中访问 http://localhost:8000
# 点击"加载规划数据"即可查看八学期课表
```

## 核心功能

| 功能 | 分值 | 说明 |
|------|------|------|
| 🔧 时间冲突处理 | 10 分 | 三维检测（周次 × 星期 × 节次），多时间段全量检查 |
| 📊 先修关系处理 | 10 分 | 邻接表 DAG + Kahn 拓扑排序 + 环检测 + 缺失先修课检测 |
| 📈 学分上限检查 | 10 分 | 贪心分配 + 不可行原因诊断 |
| 📋 类别与总学分 | 10 分 | 必修/学科基础/选修分类统计 + 选修补足 |
| 🖥️ 图形化界面 | 15 分 | 数据导入/导出/展示/筛选（7 字段组合） |
| 📖 实验报告 | 15 分 | 详细文档（目录结构/数据结构/算法流程/测试结果/复杂度分析） |
| 🎯 拓展 1 | 10 分 | 个性化时间偏好满足度分析 |
| 🖱️ 拓展 2 | 10 分 | 前端互动课表（手动选课/取消/保存/冲突查看/重新检查） |
| 🤖 拓展 3 | 10 分 | AI 选课建议（DeepSeek API + 离线回退 + SKILL 定义文档） |

## 技术栈

- **后端**：C++17, CMake（零外部依赖，内嵌 JSON 解析器）
- **前端**：HTML5, CSS3, JavaScript, ECharts 5.5
- **数据**：CSV (course_info + course_time) + JSON (constraints + major_profiles)

## 算法核心

```
Phase 0: 检查缺失先修课
Phase 1: 构建先修关系图（1000 节点 / 664 边），计算先修链深度（DFS + 记忆化）
Phase 2: 按学期迭代(1→8)，贪心分配必修课（先修链深优先 + 软约束评分选教学班）
Phase 3: 从 149 门选修候选中补足选修学分
Phase 4: 全面验证（冲突 / 先修 / 学分）
Phase 4.5: 分析个性化时间偏好满足度
Phase 5: 生成可解释的规划摘要
```

**时间复杂度**：O(N + E + T×C×O)，其中 N=1000 节点，E=664 边，T=8 学期，C=候选课数，O=教学班数。实测 < 0.5 秒。

## 拓展功能使用指南

### 拓展 1：前端互动课表

**入口**：打开 `frontend/index.html`，切换到 "📅 互动课表" Tab。

**功能清单**：

| 操作 | 方法 |
|------|------|
| 🖱️ 删除课程 | 鼠标悬停课程条目 → 点击右侧出现的 ✕ 按钮 |
| ➕ 手动添加课程 | 先在"📋 课程浏览" Tab 导入课程数据，再到互动课表底部选择课程和学期 → 点击"添加" |
| 💾 保存方案 | 点击"💾 保存当前方案"按钮 → 方案存储在浏览器本地 |
| 📂 加载方案 | 点击"📂 加载已保存方案"按钮 → 恢复之前保存的方案 |
| 🗑️ 清空方案 | 点击"🗑️ 清空方案" → 确认后清空所有学期课程 |
| ⚠️ 查看冲突 | 点击"⚠️ 查看冲突" → 弹出详细冲突列表（含具体时间段） |
| 🔍 重新检查 | 点击"🔍 重新检查" → 即时检测当前方案的时间冲突数量 |

**使用前准备**（重要）：
1. 先运行 `course_planner.exe`，确保生成了 `data/schedule_result.json`
2. 打开 `frontend/index.html`，在 "📊 规划总览" Tab 点击 **"🔄 加载规划数据"**（自动加载规划结果）
3. 如需手动添加课程，再点击 **"📥 导入课程数据"**，选择 `build/Release/data/course_dataset.json`
4. 加载完成后切换到 "📅 互动课表" Tab 即可使用

**手动添加课程须知**：系统会自动检查季节匹配（秋季课程只能加在 1/3/5/7 学期，春季课程只能加在 2/4/6/8 学期）和重复添加，不符合条件的会给出提示。

### 拓展 2：个性化时间偏好安排

系统在规划时自动为每门课程选择最符合你时间偏好的教学班（优先偏好时段、避开避让时段），规划完成后会输出每学期的偏好满足度报告。

**使用方式**：

1. 运行 `course_planner.exe`，命令行会输出类似报告：
   ```
   第1学期: 偏好满足 1/52，成功避让 49/52，综合满足率 48%
   ```
2. 在前端页面 `index.html` 中，加载规划数据后可在 "规划总览" Tab 的 "🎯 个性化时间偏好满足度" 面板查看详细报告

**如何自定义偏好**：修改 `data/sample_constraints.json` 中的 `avoid_time_blocks` 和 `preferred_time_blocks` 数组即可。示例：
```json
{ "day": "Fri", "beg": 6, "last": 4, "hard": false, "reason": "尽量避开周五下午" }
```

### 拓展 3：AI 选课顾问

**入口**：在前端任意页面点击顶部 "🤖 AI顾问" 链接，或在 `frontend/` 目录下直接打开 `ai-advisor.html`。

**两种使用模式**：

| 模式 | 说明 |
|------|------|
| 🟠 离线模拟 | 无需 API Key，点击 5 个快捷按钮（降低学分压力/避开周五下午/尽快修机器学习/评价当前方案/大三交换），系统基于关键词匹配和模板库给出建议 |
| 🟢 DeepSeek 在线 | 配置 API Key 后（存储在 `frontend/apikey.json`，已被 `.gitignore` 保护），系统将真实课程数据和当前规划结果发送给 DeepSeek 大模型，获得智能化选课建议 |

**配置 DeepSeek API Key**：
1. 将你的 API Key 写入 `frontend/apikey.json`（该文件不会被 Git 跟踪）
2. 或在 AI 顾问页面顶部的 API Key 输入框中直接输入并保存
3. 配置成功后，页面顶部会显示 "🟢 DeepSeek 在线"

**支持的自然语言示例**：
- "我希望前三学期课程压力小一些，每学期不超过18学分"
- "请评价当前方案的负载均衡性"
- "我大三要交换，大四要实习，请帮我调整"

详细的 SKILL 定义（意图识别规则、约束映射、调用管道）见 [`docs/AI_SKILL.md`](docs/AI_SKILL.md)。
